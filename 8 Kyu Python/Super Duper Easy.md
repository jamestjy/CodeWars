# Super Duper Easy

Make a function that returns the value multiplied by 50 and increased by 6. If the value entered is a string it should return "Error".

# Given Code

```{python}
def problem(a):
    #Easy Points ^_^
```

# My Solution

```{python}
def problem(a):
    #Easy Points ^_^
    if type(a) == str:
        return "Error"
    else: 
        return (a * 50) + 6
```
