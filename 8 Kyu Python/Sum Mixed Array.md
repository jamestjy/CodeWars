# Sum Mixed Array

Given an array of integers as strings and numbers, return the sum of the array values as if all were numbers.

Return your answer as a number.

# Given Code

```{python}
def sum_mix(arr):
    #your code here
```

# My Solution

```{python}
def sum_mix(arr):
    #your code here
    sum = 0
    for i in arr:
        sum += int(i)
    return sum
```
